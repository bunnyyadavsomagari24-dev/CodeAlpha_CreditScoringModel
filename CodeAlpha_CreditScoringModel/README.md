# CodeAlpha Credit Scoring Model

## Objective
Predict whether a customer has Good Credit or Bad Credit using Machine Learning.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

python generate_dataset.py
